# deno-telnet
Telnet / GMCP helper and parser for Deno.

## Usage

```ts
import { Parser, Option, Command } from "https://deno.land/x/deno_telnet@0.2.2/mod.ts";
```

Where `0.2.2` is a valid release tag or branch.

Check `examples` folder for example files.
